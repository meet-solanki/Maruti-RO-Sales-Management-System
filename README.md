# Maruti R.O. & Sales Management System

## Project Description

Maruti R.O. & Sales Management System is a web based application developed using ASP.NET and SQL Server. This system helps to manage product details, customer records, sales operations, billing system and order management.

The main purpose of this project is to convert manual business operations into a computerized system for better efficiency and accuracy.
